# Project Structure

- src/modules: screens by business area.
- src/shared/buttons: reusable buttons.
- src/shared/inputs: reusable inputs.
- src/shared/grids: reusable dynamic grids.
- src/shared/composables: reusable composition functions.
- src/shared/excel: Excel export helpers.
- src/shared/fm-icons: reusable visual elements.

Current screens are wrapped from modules first, while the original working components remain under src/components.
