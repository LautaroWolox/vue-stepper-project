# Registro OTs Fallidas

Estructura modular para la pantalla migrada.

- `components/filters`: filtros de busqueda.
- `components/grid`: grilla, paginado, exportacion y acciones.
- `constants.js`: columnas y filtros iniciales.

La pantalla actual sigue funcionando desde `src/components/certificacion/RegistroOtFallidas.vue` mientras se completa el traslado interno por subcomponentes.
